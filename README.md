# Analysis-of-Motor-Vehicle-Collisions-in-New-York-City
Data 490 Capstone Project
Authors: Ian Horn, Victor Revilla, Rene Rhyne
Arizona State University Fall 2025

Data sets
Motor Vehicle Collisions - Crashes from The City of New York is the primary data set for analysis. It is an accumulation of FORMS (Finest Online Records Management System) data, which is sourced from officers reporting traffic incidents. This data set is located on data.gov https://catalog.data.gov/dataset/motor-vehicle-collisions-crashes and it is also available from NYC Open Data https://data.cityofnewyork.us/Public-Safety/Motor-Vehicle-Collisions-Crashes/h9gi-nx95/about_data
